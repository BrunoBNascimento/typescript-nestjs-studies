# TypeScript - Nest.JS (Estudos)

Repositório exclusivo para estudos pessoais de TypeScript utilizando o framework Nest.JS

## Instalação

```bash
$ yarn install
#or
$ npm install
```

## Execução

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Execução de testes

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
