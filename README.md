# TypeScript - Nest.JS (Estudos)

Repositório exclusivo para estudos pessoais de TypeScript utilizando o framework Nest.JS

## Instalação

```bash
# use docker
# or
$ npm install
```

## Execução

```bash
# development - docker
$ docker-compose up dev

#development - npm (watch)
$ npm run dev
```

## Execução de testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
